# Llefià Music Fest
### Este lugar esta creado para fomentar y enseñar la cultura de la musica, ofreciendo unas entradas para el sitio web y dando una oportunidad para aumentar nuestra cultura musical

- Para la creación de esta pagina web se ha creado con la siguiente estructura: imagenes/ -> paginas/cartel.html, contacto.html, horario.html -> index.html, style.css y por ultimo el readme

- Para visualizar el archivo html, hay que descargar en el visual una extension llamada "Live Server", despues de instalarla, con el cursor nos situaremos encima del archivo, haremos click derecho y le daremos a abrir con Live Server
